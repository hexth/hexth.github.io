Update on Hugo

December 24, 2024

Successfully deployed Hugo on Github. Encountered some issues with Github Action but I ended up using a preset instead of following the guide. Next issue to tackle is custom domain.

I’ve recently acquired hexth.com after its registration expired and decided to keep it together with hexth.net for the time being. I then set up hexth.com for this page and encountered a few difficulties; Github cant resolve the new domain properly until I did my domain verification, this issue resolved itself once domain verification was completed and https enforced.

So now I have a blog, with custom domain, and https enabled, generated by Hugo. Quite a success I’d say.

Next thing I need to tackle is to make sure www.hexth.com resolve to hexth.com as it seems like it’s not working at the moment.
